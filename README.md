regressionuilding Insurance Claim Prediction 

## Project Overview
This project aims to predict the probability that a building will have at least one insurance claim during an insured period based on its characteristics.

## Dataset
- Train_data.csv: Main training dataset
- Variable Description.csv: Description of all variables

 ## Target Variable
 Claim:
 - 1 'n Building has at least one claim
 - 0 'n Building has no claim

## Approach
1. Data cleaning and preprocessing
2. Exploratory Data Analysis (EDA)
3. Feature engineering and encoding
4. Model building (Logistic Regression, Random Forest, Gradient Boosting)
5. Model evaluation using classification matrics
6.
